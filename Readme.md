# Mike Mignola - Fan Tribute

![Cover Image](seo-cover.png)

A fan tribute website dedicated to the legendary comic book artist and writer, Mike Mignola, creator of Hellboy.

## Features

- **Responsive Design**: Optimized for both desktop and mobile devices.
- **Internationalization**: Available in English and Spanish (auto-detected).
- **Interactive Elements**: Scroll animations and hover effects.
- **Dark Aesthetic**: Styled to match the Mignolaverse atmosphere.

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)

## How to View

Simply open `index.html` in your web browser.
